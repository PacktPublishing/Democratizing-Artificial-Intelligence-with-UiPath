Amazon Review Dataset

DESCRIPTION
The following datasets (train, test) are condensed versions of the origin Amazon reviews dataset. The files train.csv and test.csv contain two columns 1) target - a classification of the review sentiment (Very Negative, Negative, Neutral, Positive, Very Positive) and 2) input - the acutal amazon review itself 

ORGIN
The datasets included are condensed versions of the full score dataset constructed by Xiang Zhang (xiang.zhang@nyu.edu). The dataset was used as a text classification benchmark in the following paper: Xiang Zhang, Junbo Zhao, Yann LeCun. Character-level Convolutional Networks for Text Classification. Advances in Neural Information Processing Systems 28 (NIPS 2015).

The original Amazon reviews dataset (used by Xiang Zhang, and condensed for 'Democratizing Artifical Intelligence with UiPath') consists of reviews from amazon. The data span a period of 18 years, including ~35 million reviews up to March 2013. Reviews include product and user information, ratings, and a plaintext review. For more information, please refer to the following paper: J. McAuley and J. Leskovec. Hidden factors and hidden topics: understanding rating dimensions with review text. RecSys, 2013.
